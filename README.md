## Overview

This repository holds only a compiled production build (`dist/`) of the **Moderní komunikace** website — same title, same portfolio images and content as the [`moderni-komunikace`](https://github.com/dyntr/moderni-komunikace) source repo, but no editable source files. It looks like a deploy snapshot rather than an active project.

## ✦ Features

Static, pre-built HTML/CSS/JS bundle plus optimized image assets — nothing to configure or edit here.

## 🛠 Built with

Output of a Vite build (see `moderni-komunikace` for the actual React/TypeScript/Tailwind source and stack).

## 📦 Getting started

```bash
npx serve dist
```

## 🚀 Status

Archived build snapshot — for the maintained source, see [dyntr/moderni-komunikace](https://github.com/dyntr/moderni-komunikace).
